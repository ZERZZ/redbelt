# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-20

### Added

- Initial Redbelt release
- Tray-based operator workflow interface
- Configurable hotkey-driven tool launching
- Section-based tool organisation for Linux, Windows, Shells and Web workflows
- Configurable tool catalogue through `config/config.json`
- Tool management through the settings interface
- Local HTTP server for hosting payloads and scripts
- Listener management with configurable interfaces, ports and protocols
- Automatic command templating with `{IP}` and `{PORT}` placeholder substitution
- Clipboard-based command generation for rapid execution workflows
- Optional automatic shell stabilisation workflow with configurable terminal handling
- Optional automatic terminal opening on listener connections
- Startup, notification and tray behaviour configuration
- Included starter tool catalogue for common enumeration, post-exploitation and workflow tasks
